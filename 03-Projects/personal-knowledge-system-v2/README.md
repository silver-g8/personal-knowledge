## Objective

Build and maintain a clean, reusable personal knowledge system that supports daily capture, learning notes, reference material, AI-assisted workflows, and long-term review.

## Scope

- Maintain the top-level PKS folder structure.
- Keep indexes, workflows, templates, and AI rules up to date.
- Improve daily use habits.
- Add review systems that keep AI-generated content accurate and useful.

## Status

PKS v2 has a working foundation and an initial AI-assisted workflow. The next priority is turning the structure into a repeatable daily and weekly system.
