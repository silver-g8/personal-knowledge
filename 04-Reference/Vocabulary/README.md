# Vocabulary

This folder stores reusable vocabulary and glossary entries for PKS v2.

Use this folder when a word, phrase, abbreviation, or concept should be reused across multiple learning notes.

## Files

- [Vocabulary Index](vocabulary-index.md): central registry for all saved vocabulary.
- [Terms](terms/): detail notes for terms that need longer explanation.

## Rules

- Check [Vocabulary Index](vocabulary-index.md) before adding a new term.
- Use one canonical row per term.
- Add aliases to the same row or detail note instead of creating duplicates.
- Keep short meanings concise and source-aware.
- Create a detail note only when a term needs examples, context, or related links.

## Canonical ID

Use `Canonical ID` to prevent duplicates.

- English terms: lowercase and kebab-case.
- Thai terms: use the stable Thai term or a readable slug.
- Abbreviations: lowercase unless the uppercase form is necessary for meaning.

Examples:

| Term | Canonical ID |
| --- | --- |
| Margin of Safety | margin-of-safety |
| Artificial Intelligence | artificial-intelligence |
| AI | artificial-intelligence |
| ความปลอดภัยส่วนเผื่อ | ความปลอดภัยส่วนเผื่อ |
