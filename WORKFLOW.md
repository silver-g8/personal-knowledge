# Knowledge Workflow

The PKS v2 workflow is:

```text
Capture -> Organize -> Connect -> Review -> Improve -> Reuse
```

## Capture

Use `00-Inbox/` for quick capture.

Capture can include ideas, links, excerpts, rough notes, questions, or task fragments.

## Organize

Move notes into the correct folder when their purpose becomes clear.

- Learning material goes to `02-Learning/`.
- Active work goes to `03-Projects/`.
- Stable facts go to `04-Reference/`.
- Long-term responsibilities go to `05-Areas/`.
- AI workflows go to `08-AI/`.

## Connect

Add links between related notes.

Prefer relative links.

## Review

Use regular review cycles:

- Daily: update the daily note and process quick captures.
- Weekly: clear `00-Inbox/`, update project progress, and refresh task lists.
- Monthly: review areas, roadmap, templates, and archive candidates.

## Improve

Improve notes by clarifying structure, adding examples, removing duplication, and linking related material.

Do not change user intent.

## Reuse

Knowledge is useful when it can be reused in projects, decisions, writing, learning, teaching, and AI-assisted workflows.
